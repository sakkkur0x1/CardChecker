# 🌸 Go BIN Checker

A high-performance, aesthetic CLI tool to retrieve bank information from card BINs. Built with Go for speed and efficiency.

### 🚀 Usage
Just run the executable or use Go:
```bash
go run main.go
```

Or build this:
```bash
go build -ldflags="-s -w" -o cardChecker.exe main.go
```
For deleting local database type `del`.
